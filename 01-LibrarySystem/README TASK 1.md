# Task 1: Library Management System

**Objective (Summary):**  
Build a simple Library Management System to practice **OOP concepts** like classes, objects, encapsulation, and basic methods.  
The system should allow:
- Adding books to the library
- Registering members
- Borrowing and returning books
- Checking book availability

**Main Classes:**
- `Book` class: Represents a book with its properties and methods to borrow and return it.
- `Member` class: Represents library members and their borrowed books.
- `Library` class: Manages the library, adding books, registering members, lending and receiving books.
